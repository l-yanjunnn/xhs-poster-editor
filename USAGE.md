# 小红书排版编辑器 使用说明

## 怎么打开

双击 **editor.html**。会自动用浏览器（Chrome / Safari / Edge 都行）打开。

无需联网（除了首次加载霞鹜文楷字体）、无需安装。

## 操作步骤

1. **左边写文字**：在编辑区直接输入或粘贴文字，用上方工具栏切换 H1 / H2 / H3 / 引用 / 代码 / 列表 / 分隔线 / 分页符
2. **右边实时预览**：9:16 画布按你写的内容实时排版，分页符处自动换页
3. **选样式**：顶部工具栏可换主题（宣纸 / 极简白 / 深夜黑）、字体、叠色、Logo 显示策略
4. **换素材**：「🖼️ 背景库」「🐱 Logo 库」可选内置或上传你自己的
5. **保存样式**：调好的样式点「💾 保存为模板」，下次一键套用
6. **导出**：「📥 导出 PNG」→ 填文件名 → 选保存位置 → 下载 zip

## 小技巧

- **标题断行**：H1 默认占左侧 2/3，希望全宽就在工具栏改「H1 宽度 → 全宽」
- **手动换行**：在标题里按 Shift+Enter 强制换行（不另起段）
- **分页**：内容到合适位置点「⎯ 分页符 ⎯」插入分页线
- **撤销**：编辑区里 Cmd+Z

## 数据存哪儿

- 你上传的背景图、Logo、自定义模板**只存在你的浏览器里**
- 换浏览器、清缓存会丢失
- 重要素材建议保存原图，或用「保存为模板」功能记下样式
